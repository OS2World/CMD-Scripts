
msgtool
=======

- About

	msgtool is a multiplatform MSG file "decompiler".

	I wrote msgtool several years ago and I release this version now.
	
	msgtool expects the MSG file as parameter, writes the "decompiled" information in the standard output and other information in the standard error.
 
- Requirements

	Any REXX interpreter
