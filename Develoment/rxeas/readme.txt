- About.

	This is a simple set of REXX scripts to manage extended attributes in
OS/2 using the standar SysPutEA, SysGetEA and SysQueryEAList (the last of them
only available in some environments) REXXUTIL functions.

- Requirements.

REXX suport.
WPS part of Object REXX for 'lsea'?

(If you find other requirements, please mail me.)

- Files.

ico4cmd.cmd: assigns its icon to a set of CMD and ERX files without
             assigned icon.
lsea.cmd: shows the EAs from a set of files in a directory.
rmea.cmd: deletes EAs from a set of files in a directory.
rmrxea.cmd: deletes REXX EAs from a of CMD and ERX files.

- Other utilities to manage extended attributes.

noea, included in the lxLite package, by Andrew Pavel Zabolotny. It's
a powerfull tool to list and delete EAs.

EABrowse, PM program by Henk Kelder.
