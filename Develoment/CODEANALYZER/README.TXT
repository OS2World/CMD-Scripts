CodeAnalyzer is not a "user" tool.  It is a tool for REXX programmers that must
deal with large, complex programs.

This version is a minor bug fix and editorial changes release.  It is likely to 
be the last version for at least several months.

Contents of package:

CodeAnalyzer.ico               - icon
CodeAnalyzer.cmd               - program
CodeAnalyzer.cmd.AnalOCode.txt - example output file. made by analyzing itself
NormalRun.log                  - example output to screen when making above
Documentation.lwp              - source documentation
Documentation.pdf              - above in PDF format

It is released in the hope others may find it useful and that others may improve
or enhance it.
