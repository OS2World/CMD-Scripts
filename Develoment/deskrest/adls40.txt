 
 
   Adapter Support Matrix for LAN Server Version 4.0
 
 
   4 October 1994
 
 
This document contains a list of Network Adapter Cards and NDIS MAC
Drivers which are supported on LAN Server Version 4.0 Base on OS/2
uniprocessor systems.
 
-----------------------------------------------------------------------------
 
Please note that the device drivers for some of the supported
adapter cards are not delivered in the LAN Server 4.0 base product.
