-= Squirm Process Reconfiguration v1.0 =-

This script scans your system for running
squirm processes and sends them a HUP signal
so they can re-read the pattern file.

This allows you to change the pattern file
and have it used by squirm without restarting
or reconfiguring Squid in any way.
