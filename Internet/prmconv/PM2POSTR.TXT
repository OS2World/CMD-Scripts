OVERVIEW:

  The PmMail-to PostRoad Mailer Conversion Utility is a menu driven tool.
  This Utility converts PmMail folders and address books to a compatible
  PostRoad Mailer format. This Utility will not overwrite nor erase any existing
  files without user permission. If a folder or an address book exists in the
  PostRoad directories this Utility will prompt the user to enter a new
  destination name. Upon exiting this Utility, you may begin Postroad Mailer.
  All folders converted will be found in the the designated inbasket.

INSTRUCTIONS FOR USING THE UTILITY:

  1. Before you begin using the Utility there are a few items of
     information that you need to collect:

      o Your PmMail directory name (e.g., C:\PMMAIL)

      o Your PostRoad directory name (e.g., C:\POSTROAD)

      o The destination inbasket name (e.g., postsvc@Ibm.net)
 
  2. Unzip PRMCONV.ZIP into your PostRoad Mailer directory.

  3. From the command prompt type PM2POSTR <PmMail Path> to run Utility.
