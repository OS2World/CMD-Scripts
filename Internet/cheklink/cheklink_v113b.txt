       Short Description: CheckLink ver 1.13b:Create and traverse, Web-Trees.
         Long Description:
CheckLink is a free, multi-threaded, socket aware utility used to create,
verify, traverse, and index a web-tree; where "web-tree" is defined as all the 
URLs (in-line images, anchors,etc.) that are referenced in a base HTML 
document, and in html documents reachable from this base. 
CheckLink is written in REXX, and can be run as an addon for the SRE-http 
web-server, or as a standalone program.
