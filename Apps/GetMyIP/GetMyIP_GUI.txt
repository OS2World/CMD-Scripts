GetMyIP_GUI ReadMe - Version 1.0.8
==================================

GetMyIP_GUI is a GetMyIP Client which downloads your IP and displays it on your desktop
with automatic update every 5 minutes.

NAME
   GetMyIP_GUI.EXE

DESCRIPTION
   The program GetMyIP_GUI is once started. An InterNet connection must exist.  
   The IP is on the GUI and will be updated and then every 5 minutes. 
   
   GetMyIp_GUI will use default /E:ipid.shat.net /S:/iponly/ /A:body 

INSTALLATION
   Simply copy GetMyIp_GUI.* to the "GetMyIP.EXE" directory.
   !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
   ! If you update from Version < 1.0.8 !
   !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
    delete GetMyIP.INI if you have and 
    then copy the new files.

FILES
   GetMyIp_GUI.EXE - The Runtimefile
   GetMyIp.EXE     - Helperfile
